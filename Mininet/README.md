# Mininet SDN Python Scripts

Welcome to a collection of Mininet scripts designed to explore and demonstrate **Software-Defined Networking (SDN)** concepts using **Python**. This repository serves as a practical learning resource for aspiring network engineers and developers looking to understand the fundamentals of SDN through hands-on experimentation.

---

## About This Repository

This repository contains a series of Python scripts that leverage the **Mininet** network emulator. Mininet allows you to create, manage, and test custom network topologies on a single machine. By combining Mininet with Python, you can programmatically define network behavior, automate network tasks, and gain deep insights into how SDN controllers and network devices interact.

Each script in this collection is intended to:

*   **Educate:** Provide clear examples of SDN principles.
*   **Demonstrate:** Showcase practical applications of Python in network automation.
*   **Experiment:** Allow you to easily set up and test various network scenarios.

---

## Getting Started

### Prerequisites

Before running these scripts, ensure you have the following installed:

*   **Mininet:** Follow the official [Mininet installation guide](http://mininet.org/download/).
*   **Python 3:** Most modern systems come with Python 3 pre-installed.
*   **(Optional) SDN Controller:** For advanced scenarios, you might need an SDN controller like OpenDaylight, Floodlight, or onos.
